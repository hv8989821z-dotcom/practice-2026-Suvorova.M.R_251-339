# Журнал учебной практики

## Неделя 1 (30.03.2026 – 03.04.2026)
- **30.03** – Изучение технического задания. Согласование стека: Flutter, Dart, Git.
- **31.03** – Создание репозитория, клонирование, настройка веток `main`, `screen_professor`. Инициализация проекта Flutter.
- **01.04** – Разработка базовой структуры каталогов: `lib/`, `assets/`. Размещение логотипа.
- **02.04** – Проектирование навигации: нижний бар, маршруты экранов.
- **03.04** – Реализация каркаса главного экрана с `Scaffold` и `BottomNavigationBar` (заглушки).

## Неделя 2 (06.04.2026 – 10.04.2026)
- **06.04** – Верстка экрана «Преподаватели» (`teachers.dart`): хедер с логотипом, список преподавателей, кнопка поиска.
- **07.04** – Оформление стилей: цвета `rgb(33,173,252)`, `rgb(218,218,218)`, радиусы 11, шрифты, иконки пользователя.
- **08.04** – Создание переиспользуемых виджетов `Block`, `Botton` для единообразия.
- **09.04** – Экран «Профессор» (`screen_professor.dart`): вкладки «Заявки», «Чаты», «Оповещения».
- **10.04** – Добавление поиска и фильтрации. Применение стилей от экрана «Преподаватели».

## Неделя 3 (13.04.2026 – 17.04.2026)
- **13.04** – Интеграция нижнего бара `ButtonMain` в экраны преподавателя и профессора. Единый дизайн.
- **14.04** – Экран «Профиль студента»: данные, кнопки действий, обратная связь.
- **15.04** – Подключение навигации между экранами через `IndexedStack`. Тестирование переходов.
- **16.04** – Работа с Git: создание ветки `screen_professor`, коммиты, слияние с `main`.
- **17.04** – Устранение замечаний: удаление мёртвого кода, унификация цветов, фикс предупреждений.

## Неделя 4 (20.04.2026 – 24.04.2026)
- **20.04** – Финальная стилизация: скругления, отступы, типографика во всех экранах.
- **21.04** – Проверка адаптивности на разных разрешениях (`MediaQuery`).
- **22.04** – Обработка состояний: пустые списки, заглушки для неактивных кнопок.
- **23.04** – Документирование кода, комментирование ключевых методов.
- **24.04** – Подготовка отчёта и демонстрационного билда. Завершение практики.

**Используемые технологии:**  Git, Visual Studio Code.
